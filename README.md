# ipstack-cli

A simple CLI to get information about an IP address using the [ipstack](https://ipstack.com/) API.

## Installation

### Go

```sh
go install github.com/martoc/ipstack-cli@latest
```

### Docker

```sh
docker run --rm martoc/ipstack-cli:latest
```

## Usage

```sh
ipstack-cli -h
```
